# IoT Simulation: Fire Safety System

This is a simple Cisco Packet Tracer simulation demonstrating an automated fire safety system. It shows how IoT devices interact when a fire is detected to provide automated alerts and safety measures.

## 🚀 Overview

The simulation models a basic fire safety scenario:
1.  **Detection**: A Fire Detector senses fire or smoke.
2.  **Alert**: A Siren (Alarm) is triggered.
3.  **Safety**: The Garage Door opens and the Fire Sprinkler activates.

## 🛠️ Simulation Components

The simulation includes the following key items:

*   **Fire Detector**: Senses the presence of fire.
*   **Siren (Alarm)**: Visual and audible warning.
*   **Garage Door**: Opens for evacuation/ventilation.
*   **Fire Sprinkler**: suppresses the fire.
*   **Monitoring Laptop**: Used to observe the IoT device status.
*   **Network Infrastructure**: A simple router and switch to connect the devices.

## 📖 How to Run

1.  Open **Cisco Packet Tracer**.
2.  Load `fire-safety-simulation.pkt`.
3.  Move the **Fire** icon near the **Fire Detector** to trigger the system.
4.  Observe the Siren flashing, the Garage Door opening, and the Sprinkler activating.
